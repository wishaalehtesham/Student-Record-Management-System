# Student Record Management System

## Description
A console-based Student Record Management System developed in C++ as part of a C++ Programming Internship project.

## Features
- Add Student
- View Students
- Search Student
- Update Student
- Delete Student
- Save Records
- Load Records
- Input Validation

## Concepts Used
- Variables
- Functions
- Arrays
- Structures
- Loops
- File Handling
- Menu-Driven Programming

## Tools Used
- C++
- Dev C++ / Code::Blocks / VS Code

## How to Run
1. Open the project in your C++ IDE.
2. Compile the source code.
3. Run the program.
4. Follow the on-screen menu.
